# SEARCH WIZARD FAQ

## Is my data private?
**Yes.** Smart Search indexes your files locally. Only the content chunks (small snippets) are sent to Google's API for embedding or question-answering. Your actual files never leave your machine.

## What file types are supported?
- **Text:** `.pdf`, `.docx`, `.pptx`, `.txt`, `.md`, `.py`, `.js`, etc.
- **Images:** `.jpg`, `.png`, `.webp`, `.heic`
- **Video:** `.mp4`, `.mov`, `.avi`
- **Audio:** `.mp3`, `.wav`

## Does it cost money?
Smart Search itself is free and open-source. Google Gemini offers a generous **free tier** (up to 1500 requests per day) which is more than enough for personal use.

## How do I update the index?
Smart Search automatically watches for file changes in your indexed folders. If you add, delete, or modify a file, it will be updated in the search results within seconds.

## Can I index my whole hard drive?
While possible, we recommend indexing specific project or document folders for the best performance and relevance.
