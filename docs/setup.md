# SEARCH WIZARD Setup Guide

Welcome to SEARCH WIZARD! Setting up your local search is simple.

## 1. Get a Google Gemini API Key
Search Wizard requires a Gemini API key for local embeddings and assistant mode.
1. Visit [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Click **Create API Key**.
3. Copy the key.
4. Paste it into the Search Wizard onboarding screen.

## 2. Index Your Files
Indexing is the process of "reading" your files so they can be searched semantically.
1. Click the 📁 icon in the launcher.
2. Select a folder (e.g., your Documents or a Project folder).
3. The indexing progress will appear at the bottom.
4. Once complete, you can search immediately!

## 3. Keyboard Shortcuts
- **Open Launcher:** `Cmd+Shift+Space` (macOS) or `Ctrl+Shift+Space` (Windows/Linux)
- **Select Result:** Arrow Keys
- **Open File:** `Enter`
- **Preview File:** `Space`
- **Reveal in Finder:** `Cmd+R`
- **Close Launcher:** `Esc`
